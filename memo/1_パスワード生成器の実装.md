## 実装内容

### 1. 技術スタック
- **Next.js 14**（App Router）
- **TypeScript**
- **Material UI (MUI)**（`@mui/material-nextjs` で App Router 対応）
- Roboto フォント

### 2. 画面レイアウト（1920×1080 想定）
- **ナビゲーション**: 左側 360px（トップ、パスワード生成器、サンプル文字列生成、BMI計算へのリンク）
- **メイン**: 右側 1560px（コンテンツエリア）

### 3. ページ構成
| パス | 内容 |
|------|------|
| `/` | トップページ（空白） |
| `/password-generator` | **パスワード生成器**（仕様通り実装） |
| `/lorem-ipsum` | サンプル文字列生成（未実装のプレースホルダー） |
| `/bmi` | BMI計算（未実装のプレースホルダー） |

### 4. パスワード生成器の仕様
- **トグル**: 大文字・小文字・数字・アンダースコア・ハイフン
- **スライダー**: 文字数（4〜50）、生成数（1〜10）
- **「生成」ボタン**: ブラウザ内でランダム文字列を生成（サーバー送信なし）
- **表示**: 生成文字列ごとに TextField + コピーボタン
- **説明文**: 仕様書の文言をそのまま表示

### 5. ファイル構成
```
trial-2601/
├── app/
│   ├── layout.tsx          # ルートレイアウト（MUI・テーマ・Layout）
│   ├── page.tsx            # トップページ
│   ├── theme/theme.ts      # MUI テーマ
│   ├── components/Layout.tsx  # ナビ + メインのレイアウト
│   ├── password-generator/page.tsx
│   ├── lorem-ipsum/page.tsx
│   └── bmi/page.tsx
├── package.json
├── next.config.js
└── tsconfig.json
```

### 起動方法
```bash
npm run dev
```
ブラウザで http://localhost:3000 を開くと、ナビから各ページへ遷移できます。

**補足**: 現在は Next.js 14 を使用しています（React 19 との依存関係のため）。セキュリティアップデートが必要な場合は、`npm install next@latest` のうえ、必要に応じて `--legacy-peer-deps` でインストールしてください。
