# 2.1 パスワード生成器 機能の定義

## 機能名
- 日本語：パスワード生成器
- 英語：password-generator

## 概要
数種類のパラメータを選択することで、パスワードとして利用しやすいランダムな文字列を生成する。文字列生成はブラウザで行われる。

## 詳細
### 画面構成
下記の全体画面レイアウトに示す、メイン部分で機能を提供する。
![全体画面レイアウト](../images/overall_screen_layout.drawio.png)

### 機能
ユーザーは、以下のパラメータを設定した後で「生成」ボタンを押すことで、パスワード文字列を生成することができる。

#### パラメータ
| パラメータ名 | UIパーツ | 仕様 |
| :- | :- | :- |
| 大文字のアルファベットの使用 | トグルスイッチ | ONで使用する |
| 小文字のアルファベットの使用 | トグルスイッチ | ONで使用する |
| 数字 | トグルスイッチ | ONで使用する |
| アンダースコア | トグルスイッチ | ONで使用する |
| ハイフン | トグルスイッチ | ONで使用する |
| 文字数 | スライダー | 最小4, 最大50 |
| 生成数 | スライダー | 最小1, 最大10 |

#### UI

- 機能名
- 説明文章
  ```
  パスワードに使用できるランダムな文字列を生成します。
  下のスイッチやスライダーを操作することで、生成する文字列を調整することができます。
  生成はブラウザ上でのみ行われ、生成した文字列がどこかに保存されることはありません。
  ```
- 各パラメータのUI
- 「生成」ボタン
- 生成された文字列の表示エリア
  - 生成した文字列1つずつ、コピーボタン付きTextFieldで表示する
